# 👨‍🏫 教师如何用 MaxKB 备课

## 📚 教师备课痛点

- 每周备课 10+ 小时，重复劳动
- 学生提问同质化，需要重复回答
- 新教师上手慢，缺乏知识沉淀
- 教研资料散落各平台，难整合

## 🎯 MaxKB 教师备课 5 步法

### Step 1：构建学科知识库（30 分钟）

- 导入教材 PDF / Word（MaxKB 自动切片）
- 导入历年试题 / 答案
- 导入教学大纲 / 课程标准
- 导入您的备课笔记 / 教学反思

### Step 2：配置智能答疑机器人（10 分钟）

设置系统提示词：

```
你是「Python 数据分析」课程的 AI 助教。
当学生提问时：
1. 优先从知识库检索答案
2. 用通俗易懂的语言 + 代码示例
3. 如知识库没有答案，诚实告知并引导提问
```

### Step 3：课前生成教学设计（5 分钟）

用 Agent 工作流：输入教学目标 + 学情，输出教学流程 + 案例 + 练习题。

### Step 4：课中实时互动

学生扫码加入答疑群，MaxKB 自动回复基础问题。

### Step 5：课后作业批改辅助（30 分钟 → 5 分钟）

每周节省 25 小时。

## 📊 真实案例：广州华商学院 SKC4073 数据可视化课程

- 备课时间：10h/周 → 4h/周（节省 60%）
- 学生答疑响应：24 小时（之前要等下节课）
- 新生上手速度：2 周 → 3 天
- 教师满意度：⭐⭐⭐⭐⭐
