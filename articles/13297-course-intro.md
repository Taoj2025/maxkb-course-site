# MaxKB FDE 教学课程

本课程基于开源项目 **MaxKB（1Panel-dev）**，面向高校学生、教师、企业开发者，系统讲解如何将 LLM 大模型与企业知识库结合，构建真正可落地的 AI 智能体应用。

## 📚 课程内容（8 章 + 60 页 PPT）

1. MaxKB 概述与企业 AI 转型趋势
2. 核心架构：LLM + 知识库 + Agent
3. 环境搭建与开发工具链
4. 知识库构建：文档加载、切片、向量化
5. 工作流设计：函数调用、工具集成
6. Agent 编排：多轮对话、任务分解
7. 5 大企业应用场景实战
8. 项目部署、监控与运维

## 🎯 适合人群

- 高校计算机/大数据专业学生（毕设/竞赛）
- 企业 IT / 数字化转型团队
- AI 应用开发者（初级-中级）
- 教师（备课 / 教学改革）

## 📦 课程资料

- 60 页紫色 AI 科技风 PPT
- 8 章 Word 讲义（约 4 万字）
- 500+ 行 Python 核心源码
- 6 张高清配图
- 5 大真实应用场景 ROI 分析
