PART 2

6 步端到端 实战落地

从 Docker 部署到发布嵌入，整个流程 30 分钟内可完成。

1

Docker 一键部署

一条命令启动 MaxKB，开箱即用。

docker run -d --name=maxkb --restart=always \\ -p 8080:8080 -v ~/.maxkb:/opt/maxkb 1panel/maxkb
2

接入大模型

支持 DeepSeek / OpenAI / Claude / 通义千问 等 20+ 模型。

模型管理 → 添加模型 → 选择 DeepSeek base_url: https://api.deepseek.com/v1 model_name: deepseek-chat
3

创建知识库

支持 PDF / Word / Markdown / Excel / 图片 OCR 等格式上传。

知识库 → 创建 → 名称: hr_kb 向量模型: bge-large-zh-v1.5 拆分长度: 500-1000 字
4

上传文档

自动提取 → 智能拆分 → 向量化入库，一条龙处理。

解析 → 拆分 → Embedding → pgvector 状态流转：解析中 → 已索引
5

应用调试

配置开场白、提示词模板、关联知识库，在调试窗口反复测试。

简单应用 / 工作流应用 / 场景应用 3 种类型任选 · 提示词模板是关键
6

发布嵌入

iframe / JS SDK / REST API 三种方式集成到任意系统。

<iframe src="https://your-domain.com:8080/chat/embed?app_id=xxx" width="400" height="600"></iframe>
